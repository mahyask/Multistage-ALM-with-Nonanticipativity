# Multistage ALM with Nonanticipativity Constraints

This repository implements a scenario-based multistage stochastic Asset Liability Management (ALM) model using IBM CPLEX and DOcplex.

Main features:
- Scenario-based stochastic programming
- Nonanticipativity constraints
- Multistage investment decisions
- Liability management under uncertainty

Technologies:
- Python
- DOcplex
- CPLEX
